# /raw-start

Create the raw unit for the current work branch using the shared harness.

Use:

```sh
npm run harness:start -- $ARGUMENTS
```

Examples:

```sh
npm run harness:start -- --type feature --slug main-layout --title "Main layout"
npm run harness:start -- --title "Ability trigger system"
```

The canonical protocol is `docs/harness/protocols/raw-start.md`.
