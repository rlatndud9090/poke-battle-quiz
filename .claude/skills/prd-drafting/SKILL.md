# PRD Drafting 어댑터

공용 기준은 `docs/harness/protocols/prd-drafting.md`와
`docs/harness/roles/prd-writer.md`다.

PRD는 한국어로 작성한다. 문제, 목표, 비목표, 요구사항, 수용 기준, 열린 질문,
ADR 필요 여부를 모두 포함한다. 구현 세부를 확정해야 하는 사안은 PRD에 숨기지
말고 ADR 필요 항목으로 넘긴다.
