# UI Verification 어댑터

공용 기준은 `docs/harness/protocols/ui-verification.md`다.

## 확인 항목

- 모바일/데스크톱 폭에서 텍스트가 겹치지 않는가?
- playable/planned/prototype 상태가 구분되는가?
- 버튼, 탭, 입력의 의미가 드러나는가?
- shell이 특정 quiz mode에 과하게 종속되지 않는가?
- 검증 결과가 notes 또는 final report에 남는가?
