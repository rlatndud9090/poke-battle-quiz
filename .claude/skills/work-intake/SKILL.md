# Work Intake Adapter

ClaudeCode should use the shared harness:

1. Read `docs/harness/protocols/work-intake.md`.
2. Use `docs/harness/roles/intake-helper.md`.
3. Use `docs/harness/roles/unit-planner.md`.
4. If a candidate is selected, read `docs/harness/protocols/prd-drafting.md`
   and use `docs/harness/roles/prd-writer.md`.

Do not create raw units until the user accepts a candidate or clearly asks to
proceed.
