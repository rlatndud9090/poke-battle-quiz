# Work Intake 어댑터

ClaudeCode는 아래 공용 문서를 순서대로 따른다.

1. `docs/harness/protocols/work-intake.md`
2. `docs/harness/roles/intake-helper.md`
3. `docs/harness/roles/unit-planner.md`
4. 후보가 선택되면 `docs/harness/protocols/prd-drafting.md`
5. `docs/harness/roles/prd-writer.md`

사용자가 후보를 승인하거나 명확히 진행을 지시하기 전에는 raw unit을 만들지 않는다.
모든 제안과 PRD 초안은 한국어로 작성한다.
