# Unit Planner

Owns work-unit sizing and branch naming.

Responsibilities:

- Split broad ideas into independently shippable units.
- Choose `feature`, `bugfix`, or `chore`.
- Propose branch names that describe the core work.
- Keep branch slugs kebab-case and specific.
- Define in-scope and out-of-scope boundaries.

The unit planner should protect the project from oversized feature branches.
